Game About

About This Project

Chess vs GPT-4 is an interactive web-based chess game that allows players to compete against an AI powered by OpenAI's GPT-4 language model. How many moves does it take to win?

Creators

  • Matt Ovesen - Burlington High School Senior
  • Kaustubh Kislay - Burlington High School Senior

Features

  • Play chess against GPT-4 AI
  • Interactive chessboard with move highlighting
  • Chat functionality to ask questions about the game
  • Strategic AI moves considering captures, checks, and long-term positioning

This project demonstrates the potential of large language models in game playing and decision-making tasks. It combines traditional game logic with advanced AI capabilities to create a unique chess-playing experience.

Created for the InspiritAI Scholars Program 2024, this project showcases the intersection of artificial intelligence and classic game strategy.
